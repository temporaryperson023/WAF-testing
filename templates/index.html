WAF Test Lab

A testing environment for Haltdos WAF. Two lab modules — each with a different category of testing targets.

01
🛡 Vulnerability Lab
Test SQLi, XSS, SSRF, File Upload, Command Injection and Broken Access Control against the WAF using curated attack payloads.
/vuln/ ↗
02
⚙ Rules Test Lab
Verify WAF rules behavior — test header rules, redirect rules, capture rules, transform rules, and more.
/rules/ ↗

Usage — Place a WAF in front of this application (Nginx + ModSecurity, AWS WAF, or Haltdos). Send traffic through the WAF and verify detection, logging, and blocking behavior. This application performs no input validation on its own.
