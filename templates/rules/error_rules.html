Rule Type — Error Rule

Error Rules

Select any HTTP error code and fire a request. The backend returns a raw JSON error with that exact status code. Configure an Error Rule in your WAF to intercept the response and replace it with a custom error page.

Trigger Request
Select Error Code
404 Not Found 4xx Client Error
GET /rules/error/test?code=404

Hits the endpoint with the selected code. Without WAF you'll see raw JSON. With an Error Rule active the WAF should replace the response.

Response 
No request sent yet
Endpoint behaviour — /rules/error/test?code=404 always returns the selected HTTP status code with a JSON body. If you see HTML after sending, your WAF Error Rule is intercepting and replacing the response correctly.
