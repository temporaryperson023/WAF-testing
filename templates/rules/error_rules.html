Rule Type — Error Rule

Error Rules

This endpoint returns a raw 404 JSON error response from the backend. Configure an Error Rule in your WAF to intercept this status code and replace the response with a custom error page or message.

Trigger Request
GET /rules/error/test

Hits the error endpoint. Without WAF you'll see raw JSON. With an Error Rule active the WAF should replace the response.

Response 
No request sent yet
Endpoint behaviour — /rules/error/test always returns HTTP 404 with a JSON body. The raw backend never serves HTML — if you see HTML after sending, your WAF Error Rule is working correctly.
