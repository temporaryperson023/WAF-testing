Rule Type — Variable Rule

Variable Rules

Send query parameters user and role to the backend and verify the WAF modifies their values before they arrive. Without a rule the backend echoes back exactly what you sent.

Send Request
GET /rules/variable/test?user=&role=

Values are sent as query params. If the WAF Variable Rule is active, the backend will receive modified values — not what you typed.

Backend Received 
No request sent yet
What to look for — If the WAF Variable Rule is active, the received_user or received_role values in the response will differ from what you entered in the fields above.

Without WAF Backend echoes sent values
With WAF Backend receives modified values
